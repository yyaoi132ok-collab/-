# 姚羿数据分析作品集网站

## 本地运行

```bash
npm install
npm run dev
```

打开终端显示的本地地址，默认是 `http://127.0.0.1:4173/`。

## 常用命令

```bash
npm run check
npm run build
npm run preview
```

- `check` 检查作品集文案、资源与关键视觉规则。
- `build` 生成生产版本到 `dist/`。
- `preview` 预览已构建的生产版本。

## 目录说明

- `src/` React 页面与样式。
- `public/` 项目图表、地图数据、校徽与头像资源。
- `scripts/` 内容检查脚本。
- `dist/` 当前已构建的可部署静态文件。
